# Space 枫叶导航站

这是一个免费可部署的静态网址导航页，风格参考 hao123，但界面更简洁、更有科技感。

## 当前功能
- 百度搜索框
- 分类网址导航
- 响应式布局（电脑/手机可用）
- 纯 HTML + CSS，无需后端，免费托管

## 免费部署方案
### 方案 1：GitHub Pages
1. 新建 GitHub 仓库
2. 上传 `index.html`
3. 在 Pages 中选择从 main 分支部署
4. 获得免费网址

### 方案 2：Cloudflare Pages
1. 登录 Cloudflare
2. 创建 Pages 项目
3. 上传本目录
4. 自动得到免费域名，可后续绑自定义域名

### 方案 3：Netlify
1. 登录 Netlify
2. 拖拽本目录上传
3. 自动发布静态网站
